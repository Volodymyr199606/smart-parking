# Smart Parking — Website

Next.js marketing website for the Smart Parking product.

## Purpose

- Landing page (product overview, features, CTA)
- Interactive demo (web preview of the mobile app experience)
- Waitlist signup
- Subscription / pricing (future)
- Privacy policy and terms

## Tech Stack

- Next.js 14+ (App Router)
- TypeScript
- Tailwind CSS
- Supabase client (for waitlist and future auth)
- Deployed on Vercel
