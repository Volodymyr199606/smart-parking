# Smart Parking — Mobile App

React Native Expo app for iOS and Android. This is the main product.

## Purpose

- Find available street parking on a real-time map
- Navigate to open spots
- Report spot availability
- Save favorite parking locations
- User authentication

## Tech Stack

- React Native + Expo (managed workflow)
- TypeScript
- Expo Router (file-based navigation)
- react-native-maps (Apple Maps / Google Maps)
- Supabase client (auth, database, realtime)
- Deployed via Expo EAS
